# 🧠 Early Detection of Cognitive Decline due to Alcohol

## 📌 Overview
This project focuses on the **early detection of cognitive decline** caused by alcohol consumption using a combination of **behavioral assessment** and **EEG signal analysis**. The system integrates **machine learning techniques** with **brain signal processing (FFT)** to provide an objective and interpretable prediction of cognitive health.

---

## 🎯 Key Features
- 🧾 Behavioral questionnaire-based risk assessment  
- 🧠 EEG signal analysis using Fast Fourier Transform (FFT)  
- 🤖 Machine Learning models (Logistic Regression, SVM, Random Forest, XGBoost)  
- 📊 Explainable AI using SHAP for feature importance  
- ⚠️ Early risk detection before clinical symptoms appear  
- 📄 Personalized cognitive health report generation  

---

## ⚙️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js / Python (Flask)  
- **Machine Learning:** Scikit-learn, Logistic Regression , Random Forest, XGBoost  
- **Data Processing:** NumPy, Pandas  
- **Visualization:** Matplotlib, Seaborn  
- **Explainability:** SHAP  

---

## 🔄 Workflow
1. User fills behavioral questionnaire  
2. Risk score is generated  
3. High-risk users are recommended EEG analysis  
4. EEG signals are processed using FFT (time → frequency domain)  
5. ML models classify cognitive state (Healthy / At-Risk)  
6. Results and insights are displayed to the user  

---

## 📊 Results
- Logistic Regression achieved **82.7% accuracy** for early detection  
- Model shows **strong generalization** due to limited dataset size  
- Multimodal approach improves **real-world applicability**
